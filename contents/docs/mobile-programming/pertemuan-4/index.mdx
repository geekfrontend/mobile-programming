### ðŸ§  Komponen Dasar & Props Penting

#### 1. View â€“ Container Layout

```tsx
import { View } from "react-native";

<View style={{ padding: 16, backgroundColor: "#eee", borderRadius: 8 }}>
  <Text>Inside a View</Text>
</View>
```

Props berguna: `style`, `pointerEvents`, `onLayout` (membaca ukuran komponen).

#### 2. Text â€“ Menampilkan Teks

```tsx
import { Text } from "react-native";

<Text style={{ fontSize: 18, fontWeight: "bold" }}>
  Hello React Native
</Text>
```

Gunakan `numberOfLines` untuk membatasi baris, `ellipsizeMode` untuk mengatur pemotongan teks.

#### 3. Image â€“ Gambar Lokal & Remote

```tsx
import { Image } from "react-native";

<Image
  source={{ uri: "https://ui.shadcn.com/avatars/02.png" }}
  style={{ width: 80, height: 80, borderRadius: 40 }}
  resizeMode="cover"
/>
```

Asset lokal: `source={require("../assets/icon.png")}`. Perhatikan ratio agar gambar tidak gepeng.

#### 4. TextInput â€“ Input Pengguna

```tsx
import { useState } from "react";
import { TextInput } from "react-native";

const [name, setName] = useState("");

<TextInput
  value={name}
  onChangeText={setName}
  placeholder="Enter your name"
  style={{ borderWidth: 1, borderColor: "#ccc", padding: 12, borderRadius: 8 }}
/>
```

Props penting: `keyboardType`, `secureTextEntry`, `autoCapitalize`, `onSubmitEditing`.

#### 5. Button â€“ Action Sederhana

```tsx
import { Button } from "react-native";

<Button title="Simpan" onPress={() => alert("Button pressed!")} />
```

Untuk tampilan custom gunakan `Pressable` atau `TouchableOpacity` agar bisa di-styling bebas.

#### 6. ScrollView â€“ Konten Panjang

```tsx
import { ScrollView, Text } from "react-native";

<ScrollView contentContainerStyle={{ padding: 16 }}>
  {Array.from({ length: 12 }, (_, i) => (
    <Text key={i}>Item {i + 1}</Text>
  ))}
</ScrollView>
```

Gunakan ketika konten lebih tinggi dari layar. Untuk list panjang gunakan `FlatList` (akan dibahas pertemuan selanjutnya).

### ðŸ§© Studi Kasus: Profile Card

```tsx
import { useState } from "react";
import {
  View,
  Text,
  Image,
  TextInput,
  Button,
  ScrollView,
  StyleSheet,
  Alert,
} from "react-native";

export default function App() {
  const [status, setStatus] = useState("");

  const handleSave = () => {
    if (!status) {
      Alert.alert("Oops", "Status tidak boleh kosong");
      return;
    }
    Alert.alert("Status", `Status tersimpan: ${status}`);
  };

  return (
    <ScrollView contentContainerStyle={styles.container}>
      <Text style={styles.header}>My Profile Card</Text>

      <View style={styles.card}>
        <Image
          source={{ uri: "https://ui.shadcn.com/avatars/02.png" }}
          style={styles.avatar}
        />
        <Text style={styles.name}>John Doe</Text>
        <Text style={styles.role}>Mobile Developer</Text>
        <Text style={styles.description}>
          Passionate about crafting delightful mobile experiences using React
          Native and TypeScript.
        </Text>

        <TextInput
          value={status}
          onChangeText={setStatus}
          placeholder="Update status..."
          style={styles.input}
        />

        <Button title="Save Status" onPress={handleSave} />
      </View>
    </ScrollView>
  );
}

const styles = StyleSheet.create({
  container: { flexGrow: 1, padding: 24, backgroundColor: "#fff" },
  header: { fontSize: 24, fontWeight: "bold", marginBottom: 24 },
  card: {
    alignItems: "center",
    backgroundColor: "#f9f9f9",
    padding: 24,
    borderRadius: 16,
    gap: 12,
    shadowColor: "#000",
    shadowOpacity: 0.1,
    shadowRadius: 5,
    elevation: 3,
  },
  avatar: { width: 100, height: 100, borderRadius: 50 },
  name: { fontSize: 20, fontWeight: "bold" },
  role: { fontSize: 16, color: "#555" },
  description: { textAlign: "center", color: "#666" },
  input: {
    borderWidth: 1,
    borderColor: "#ccc",
    width: "100%",
    padding: 12,
    borderRadius: 8,
    backgroundColor: "#fff",
  },
});
```

> Coba ubah ukuran font, warna, dan `borderRadius` untuk melihat efeknya langsung melalui hot reload.

### ðŸ§ª Latihan Terarah

1. Duplikasi komponen profile card dan tampilkan dalam `ScrollView` untuk mensimulasikan daftar anggota tim.  
2. Ganti komponen `Button` dengan `TouchableOpacity` custom yang memiliki variasi warna (primary, secondary).  
3. Tambahkan validasi pada `TextInput` agar minimal 10 karakter.  
4. Tambahkan properti `keyboardShouldPersistTaps="handled"` pada `ScrollView` dan jelaskan kegunaannya.  
5. (Opsional) Gunakan `KeyboardAvoidingView` agar form tidak tertutup keyboard pada perangkat iOS/Android.
