## ğŸ“Œ Pertemuan 1: Pengantar TypeScript

TypeScript adalah **superset dari JavaScript** yang dikembangkan oleh **Microsoft**.  
Semua kode JavaScript valid merupakan kode TypeScript, namun TypeScript menambahkan fitur seperti **static type checking**, **interface**, dan **fitur OOP modern** sehingga kode lebih terstruktur dan mudah dirawat.

- Mengompilasi proyek TypeScript menjadi JavaScript menggunakan `tsc`.  
- Menjalankan latihan kecil untuk membiasakan penulisan tipe data dasar.

### ğŸ§  Konsep Utama

#### 1. Type Annotation & Type Inference

```ts
let name: string = "John";
let age: number = 21;
let isActive: boolean = true;

function hello(user: string): string {
  return `Hello, ${user}!`;
}

console.log(hello(name));
```

â¡ï¸ Ketika tipe tidak disebutkan, TypeScript berusaha **menginfersikan**. Namun pada latihan awal, biasakan menuliskan tipe secara eksplisit agar lebih mudah dibaca.

#### 2. Struktur Data Dasar

| Tipe          | Contoh                                                      |
| ------------- | ----------------------------------------------------------- |
| `string`      | `"Hello World"`                                             |
| `number`      | `123`, `3.14`                                               |
| `boolean`     | `true`, `false`                                             |
| `array`       | `let numbers: number[] = [1, 2, 3];`                        |
| `tuple`       | `let point: [number, number] = [0, 1];`                     |
| `object`      | `let user: { name: string; age: number } = { ... }`         |
| `any`         | Hindari kecuali benar-benar butuh fleksibilitas penuh.      |

#### 3. Function & Optional Parameter

```ts
function calculateSquareArea(side: number, unit?: string): string {
  const area = side * side;
  return unit ? `${area} ${unit}Â²` : area.toString();
}

calculateSquareArea(4); // "16"
calculateSquareArea(4, "cm"); // "16 cmÂ²"
```

Parameter dengan `?` bersifat opsional. Saat menggunakan type annotation, TypeScript langsung memberi tahu bila argumen yang dikirim tidak sesuai.

### ğŸ› ï¸ Workflow Kompilasi

1. Buat file `hello.ts`:

   ```ts
   let message: string = "Hello TypeScript!";
   console.log(message);
   ```

2. Kompilasi ke JavaScript (akan menghasilkan `hello.js`):
   ```bash
   tsc hello.ts
   ```

3. Jalankan file JavaScript yang dihasilkan:
   ```bash
   node hello.js
   ```

4. Untuk proyek yang lebih besar, buat konfigurasi default dengan:
   ```bash
   tsc --init
   ```
   Perintah tersebut membuat `tsconfig.json` sehingga kita bisa menjalankan `tsc --watch` untuk kompilasi otomatis.
