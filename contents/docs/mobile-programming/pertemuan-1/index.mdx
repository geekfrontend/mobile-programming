## ğŸ“Œ Pertemuan 1: Pengantar TypeScript

TypeScript adalah **superset dari JavaScript** yang dikembangkan oleh **Microsoft**.  
Artinya, semua kode JavaScript valid juga merupakan kode TypeScript, tetapi TypeScript menambahkan fitur tambahan seperti **type system** dan **fitur OOP modern**.

Dengan TypeScript, developer dapat menulis kode yang lebih aman, lebih mudah dipahami, dan lebih terstruktur.

---

### ğŸ”‘ Kenapa Belajar TypeScript?

- **Type Checking** â†’ membantu mendeteksi error lebih awal sebelum dijalankan.  
- **Produktivitas meningkat** berkat fitur autocomplete dan IntelliSense di VS Code.  
- **Lebih terstruktur** saat mengembangkan aplikasi skala besar.  
- Mendukung fitur modern seperti *interface*, *enum*, *generic*, dan *decorator*.  
- Bisa digunakan di **frontend (React, Vue, Angular)** maupun **backend (Node.js, Deno, Bun)**.

### ğŸ› ï¸ Kompilasi TypeScript ke JavaScript

   ```bash

   ```

2. Buat file `hello.ts`:

   ```bash

   ```

4. Jalankan:
