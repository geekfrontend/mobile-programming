TypeScript adalah **superset dari JavaScript** yang menambahkan fitur **static typing**. Dengan TypeScript, kode menjadi lebih aman, terstruktur, dan mudah dipelihara.

### Apa itu TypeScript?

- Dikembangkan oleh **Microsoft**.
- Menambahkan **type system** ke JavaScript.
- Membantu menghindari bug dengan mengecek tipe data saat menulis kode.
- Bisa digunakan di **frontend** (React, Vue, Angular) maupun **backend** (Node.js).
