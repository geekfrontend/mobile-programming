## ğŸ“Œ Pertemuan 3: Membuat Project dengan Expo

Setelah memahami dasar TypeScript, kita akan mulai membuat project **React Native** menggunakan **Expo**.  
Expo adalah framework yang mempermudah pembuatan aplikasi mobile dengan React Native.  
Dengan Expo, kita bisa langsung membuat dan menjalankan aplikasi tanpa perlu konfigurasi rumit seperti Android Studio atau Xcode di awal.

---

### ğŸ”¹ Membuat Project Baru dengan Expo

Gunakan perintah berikut untuk membuat project baru:

```bash
npx create-expo-app MyApp
```

Setelah selesai, masuk ke folder project:

```bash
cd MyApp
```

Lalu jalankan aplikasi:

```bash
npm start
```

atau:

```bash
npx expo start
```

Expo akan menampilkan **QR Code** di terminal/browser yang bisa dipindai menggunakan aplikasi **Expo Go** di Android/iOS.
