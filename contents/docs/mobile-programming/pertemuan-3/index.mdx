### ğŸš€ Membuat Project Baru

Gunakan perintah berikut untuk membuat proyek baru dengan template TypeScript:

```bash
npx create-expo-app@latest expo-app --template
```

Pada prompt pilih `blank (TypeScript)` agar langsung mendapatkan dukungan TypeScript.

Setelah proses selesai:

```bash
cd expo-app
npm install # memastikan dependency awal terpasang
```

Untuk menjalankan proyek:

```bash
npm run start
```

atau langsung via Expo CLI:

```bash
npx expo start
```

Expo CLI akan membuka DevTools di browser dan menampilkan QR code untuk di-scan menggunakan Expo Go atau menjalankan emulator.

### ğŸ§ª Latihan Terarah

1. Buat proyek baru dengan nama `MyFirstExpo`.  
2. Jalankan perintah `npm run start` dan hubungkan ke Expo Go/emulator.  
3. Ubah teks di `App.tsx` menjadi komponen profile sederhana yang menampilkan nama, jurusan, dan universitas.  
4. Tambahkan file baru di `assets/` (misalnya foto profil) dan tampilkan menggunakan komponen `Image`.  
5. (Opsional) Aktifkan **Expo Router** dengan menjalankan `npx expo install expo-router react-native-safe-area-context react-native-screens`, lalu buat halaman tambahan di folder `app/`.
