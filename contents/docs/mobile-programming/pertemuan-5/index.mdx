Styling adalah bagian penting dalam pengembangan aplikasi mobile untuk membuat tampilan yang menarik dan user-friendly.  
React Native menyediakan **StyleSheet** sebagai cara utama untuk styling, dan kita juga bisa menggunakan **NativeWind** (Tailwind CSS untuk React Native).

---

### ðŸŽ¨ StyleSheet - Cara Tradisional

StyleSheet adalah API bawaan React Native untuk membuat style yang teroptimasi.

```jsx
import React from 'react';
import { View, Text, StyleSheet } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
      <Text style={styles.title}>Hello World</Text>
      <Text style={styles.subtitle}>Welcome to React Native</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#f0f0f0',
  },
  title: {
    fontSize: 24,
    fontWeight: 'bold',
    color: '#333',
    marginBottom: 10,
  },
  subtitle: {
    fontSize: 16,
    color: '#666',
  },
});
```

### ðŸ”¹ Flexbox di React Native

React Native menggunakan **Flexbox** sebagai sistem layout utama:

```jsx
const styles = StyleSheet.create({
  container: {
    flex: 1,
    flexDirection: 'column', // 'row' | 'column'
    justifyContent: 'center', // 'flex-start' | 'center' | 'flex-end' | 'space-between'
    alignItems: 'center', // 'flex-start' | 'center' | 'flex-end' | 'stretch'
  },
  box: {
    width: 100,
    height: 100,
    backgroundColor: 'blue',
    margin: 10,
  },
});
```

**Properti Flexbox Penting:**
- `flex: 1` â†’ mengisi ruang yang tersedia
- `flexDirection` â†’ arah layout (row/column)
- `justifyContent` â†’ alignment di main axis
- `alignItems` â†’ alignment di cross axis

### ðŸŒŸ NativeWind - Tailwind untuk React Native

NativeWind memungkinkan kita menggunakan **Tailwind CSS** di React Native dengan sintaks yang familiar.

#### Instalasi NativeWind

```bash
npm install nativewind
npm install --save-dev tailwindcss
```

Buat file `tailwind.config.ts`:

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["./App.{js,jsx,ts,tsx}", "./src/**/*.{js,jsx,ts,tsx}"],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

Update `babel.config.ts`:

```js
module.exports = function (api) {
  api.cache(true);
  return {
    presets: ['babel-preset-expo'],
    plugins: ["nativewind/babel"],
  };
};
```

```jsx
      </Text>

```

**Keuntungan NativeWind:**
- Sintaks familiar bagi developer web
- Utility-first approach
- Konsisten dengan Tailwind CSS
- IntelliSense support

### ðŸŽ¯ Studi Kasus: Card Component

Mari buat komponen Card yang stylish:

```jsx
import React from 'react';
import { View, Text, Image, TouchableOpacity } from 'react-native';

const ProductCard = ({ product }) => {
  return (
    <TouchableOpacity className="bg-white rounded-lg shadow-md m-2 overflow-hidden">
      <Image 
        source={{ uri: product.image }} 
        className="w-full h-48"
        resizeMode="cover"
      />
      <View className="p-4">
        <Text className="text-lg font-bold text-gray-800 mb-2">
          {product.name}
        </Text>
        <Text className="text-gray-600 mb-3">
          {product.description}
        </Text>
        <View className="flex-row justify-between items-center">
          <Text className="text-xl font-bold text-blue-600">
            Rp {product.price.toLocaleString()}
          </Text>
          <TouchableOpacity className="bg-blue-500 px-4 py-2 rounded-md">
            <Text className="text-white font-semibold">Beli</Text>
          </TouchableOpacity>
        </View>
      </View>
    </TouchableOpacity>
  );
};
```
