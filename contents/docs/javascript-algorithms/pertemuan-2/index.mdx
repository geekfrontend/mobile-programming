Struktur kontrol digunakan untuk mengatur alur program berdasarkan kondisi tertentu.

### `if`, `else if`, dan `else`

Digunakan untuk menjalankan kode berdasarkan kondisi:

```js
let nilai = 85;
if (nilai >= 90) {
  console.log("A");
} else if (nilai >= 80) {
  console.log("B");
} else {
  console.log("C");
}
```

### Operator Ternary (`? :`)

Operator ternary adalah cara singkat untuk menulis `if...else`:

```js
let status = nilai >= 80 ? "Lulus" : "Gagal";
console.log(status);
```

### `switch case`

Digunakan untuk menangani banyak kondisi dengan lebih rapi:

```js
let hari = "Senin";
switch (hari) {
  case "Senin":
    console.log("Hari kerja");
    break;
  case "Sabtu":
  case "Minggu":
    console.log("Hari libur");
    break;
  default:
    console.log("Hari tidak valid");
}
```
