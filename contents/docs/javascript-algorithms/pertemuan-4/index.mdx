Fungsi adalah blok kode yang dapat digunakan kembali untuk melakukan tugas tertentu. Di JavaScript, fungsi membantu mengorganisir dan mempermudah pemeliharaan kode.

### ğŸ“š Jenis Fungsi

1. **Fungsi Deklarasi**
   Fungsi yang dideklarasikan menggunakan kata kunci `function`.

```js
function sapa(nama) {
  console.log(`Halo, ${nama}!`);
}

sapa("Budi");
```

2. **Fungsi Ekspresi**
   Fungsi yang disimpan dalam variabel.

```js
const sapa = function (nama) {
  console.log(`Halo, ${nama}!`);
};

sapa("Ani");
```

3. **Arrow Function**
   Sintaks lebih ringkas untuk fungsi anonim.

```js
const sapa = (nama) => {
  console.log(`Halo, ${nama}!`);
};

sapa("Citra");
```

### ğŸ¯ Scope (Cakupan Variabel)

1. **Global Scope**: Variabel yang dideklarasikan di luar fungsi.

```js
let globalVar = "Saya global";

function tampilkan() {
  console.log(globalVar);
}

// Output: Saya global
tampilkan();
```

2. **Local Scope**: Variabel di dalam fungsi hanya bisa diakses di dalam fungsi tersebut.

```js
function lokal() {
  let localVar = "Saya lokal";
  console.log(localVar);
}

// Output: Saya lokal
lokal();
// Error: localVar is not defined
// console.log(localVar);
```

### ğŸ” Closures

Fungsi dalam JavaScript memiliki akses ke variabel di lingkup induknya, bahkan setelah induknya selesai dieksekusi.

```js
function induk() {
  let nama = "Joko";
  return function anak() {
    console.log(nama);
  };
}

const panggilAnak = induk();
panggilAnak(); // Output: Joko
```
