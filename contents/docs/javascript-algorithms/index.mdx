## ðŸ“Œ Pertemuan 2: Struktur Kontrol - Percabangan

Struktur kontrol digunakan untuk mengatur alur program berdasarkan kondisi tertentu. Materi yang akan dipelajari:

- **`if`, `else if`, dan `else`**
- **Operator ternary (`? :`)**
- **`switch case`** untuk banyak kondisi

### Contoh:

```js
let nilai = 85;
if (nilai >= 90) {
  console.log("A");
} else if (nilai >= 80) {
  console.log("B");
} else {
  console.log("C");
}
```

## ðŸ“Œ Pertemuan 3: Struktur Kontrol - Perulangan (Loops)

Loop digunakan untuk mengeksekusi kode berulang kali. Kita akan membahas:

- **`for`, `while`, dan `do...while`** loops
- **Nested loops** (perulangan bersarang)
- **`break` dan `continue`** untuk mengontrol loop

### Contoh:

```js
for (let i = 1; i <= 5; i++) {
  console.log("Iterasi ke-" + i);
}
```

## ðŸ“Œ Pertemuan 4: Fungsi (Functions) & Scope

Fungsi adalah blok kode yang dapat digunakan kembali. Topik yang dibahas:

- **Deklarasi fungsi (`function` vs arrow function)**
- **Parameter dan return value**
- **Scope variabel (global vs local)**
- **Closures dalam JavaScript**

### Contoh:

```js

  return `Halo, ${nama}!`;
}

console.log(sapa("Budi"));
```
