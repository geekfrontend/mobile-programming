JavaScript adalah bahasa pemrograman yang digunakan untuk mengembangkan aplikasi web interaktif. Pada pertemuan ini, kita akan membahas dasar-dasar JavaScript.

### Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang berjalan di browser maupun di server menggunakan Node.js. JavaScript sering digunakan untuk:

- Membuat halaman web interaktif
- Mengontrol elemen HTML dan CSS
- Berkomunikasi dengan server (API)
- Membangun aplikasi mobile dan desktop

### Cara Kerja JavaScript (Client-side vs Server-side)

JavaScript bisa dijalankan di dua lingkungan:

1. **Client-side**: Kode JavaScript berjalan di browser, digunakan untuk mengubah tampilan halaman web secara dinamis.
2. **Server-side**: JavaScript digunakan di backend menggunakan Node.js untuk mengelola database, autentikasi, dan lebih banyak lagi.

### Variabel dan Tipe Data

Variabel adalah tempat untuk menyimpan data. JavaScript memiliki tiga cara untuk mendeklarasikan variabel:

```js
var nama = "Budi"; // Global scope
let umur = 25; // Block scope
const PI = 3.14; // Tidak bisa diubah
```

Tipe data yang umum digunakan:

- **String** (`"Hello"`)
- **Number** (`123, 3.14`)
- **Boolean** (`true, false`)
- **Array** (`[1, 2, 3]`)
- **Object** (`{ nama: "Budi", umur: 25 }`)

### Operator Dasar

Operator dalam JavaScript meliputi:

- **Aritmatika** (`+`, `-`, `*`, `/`, `%`)
- **Perbandingan** (`==`, `===`, `!=`, `!==`, `>`, `<`, `>=`, `<=`)
- **Logika** (`&&`, `||`, `!`)
