### ğŸ“š Apa itu Array?

Array adalah struktur data yang berisi koleksi item, di mana setiap item diidentifikasi dengan indeks atau kunci. Dalam JavaScript, array dapat berisi berbagai jenis data (angka, string, objek, dll.).

```javascript
const numbers = [1, 2, 3, 4, 5];
const mixedArray = [1, "apple", true, { key: "value" }];
console.log(numbers);
console.log(mixedArray);
```

Anda dapat mendeklarasikan array dengan beberapa cara:

```javascript
// Menggunakan array literal
const fruits = ["apple", "banana", "cherry"];

// Menggunakan Array constructor

// Array kosong

```

### ğŸ› ï¸ Metode Dasar Array

1. **Menambah dan menghapus elemen:**

```javascript
const colors = ["red", "blue"];

// Menambah elemen
colors.push("green"); // Menambah di akhir
colors.unshift("yellow"); // Menambah di awal

// Menghapus elemen
colors.pop(); // Menghapus elemen terakhir
colors.shift(); // Menghapus elemen pertama

console.log(colors);
```

2. **Splicing dan slicing:**

```javascript
const animals = ["dog", "cat", "rabbit", "elephant"];

// splice(start, deleteCount, item1, item2, ...)
animals.splice(2, 0, "fox"); // Menambah 'fox' di indeks 2

// slice(start, end)
const someAnimals = animals.slice(1, 3); // Mengembalikan ['cat', 'fox']

console.log(animals);
console.log(someAnimals);
```

3. **Iterasi melalui array:**

```javascript
const scores = [10, 20, 30, 40];

// forEach
scores.forEach((score) => console.log(score));

// map (mengembalikan array baru)
const doubled = scores.map((score) => score * 2);
console.log(doubled);

// filter (mengembalikan elemen yang memenuhi kondisi)
const highScores = scores.filter((score) => score >= 30);
console.log(highScores);
```

4. **Menggabungkan nilai array:**

```javascript
const numbers = [1, 2, 3, 4, 5];

const sum = numbers.reduce((total, num) => total + num, 0);
console.log(sum); // Output 15
```

5. **Mencari dan memeriksa elemen:**

```javascript
const fruits = ["apple", "banana", "cherry"];

console.log(fruits.includes("banana")); // true
console.log(fruits.indexOf("cherry")); // 2
```

6. **Mengurutkan array:**

```javascript
const points = [40, 100, 1, 5, 25, 10];

// Mengurutkan secara ascending
points.sort((a, b) => a - b);

// Mengurutkan secara descending
points.sort((a, b) => b - a);

console.log(points);
```
